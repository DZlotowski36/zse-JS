Rodzaje pętli w programowaniu

1. Pętla for

Nazwa pętli

Pętla for

Konstrukcja

for(inicjalizacja; warunek; zmiana)
{
    instrukcje;
}

Opis

Pętla for jest używana wtedy, gdy wiadomo ile razy ma zostać wykonany dany fragment programu.

2. Pętla while (z kontrolowanym wejściem)

Nazwa pętli

Pętla while

Konstrukcja

while(warunek)
{
    instrukcje;
}

Opis

Pętla while najpierw sprawdza warunek. Jeśli warunek jest spełniony, wykonywane są instrukcje. Dlatego jest to pętla z kontrolowanym wejściem.

Powrót do warunku

3. Pętla do while (z kontrolowanym wyjściem)

Nazwa pętli

Pętla do while

Konstrukcja

do
{
    instrukcje;
}
while(warunek);

Opis

Pętla do while wykonuje instrukcje co najmniej jeden raz, ponieważ warunek sprawdzany jest na końcu. Jest to pętla z kontrolowanym wyjściem.

  |
Instrukcje
  |
Warunek?
 /   \

 |     |
